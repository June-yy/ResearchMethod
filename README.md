# Research Methods Spring Coursework 2: Data Visualization
This repository contains code and outputs for a radar chart comparing environmental impacts of different diet groups.  
Language: Python  
Libraries: matplotlib, pandas

# Files
- `ResearchMethod_CW2.ipynb`: main analysis and visualization code  
- `Results_21Mar2022.csv`: source dataset  
- `radar_chart_env_impact.png`: final output image 

To view the notebook, open it in Jupyter or VS Code with a Python environment.
